ผมชื่อ นายณัชพล แก้วตีนแท่น (นัท) อายุ 20 ปี เกิดวันที่ 6 ตุลาคม 2548 สำเร็จการศึกษาระดับ ปวส. สาขานักพัฒนาซอฟต์แวร์ จากวิทยาลัยเทคนิคขอนแก่น ด้วย GPAX 3.86

แรงบันดาลใจมาจากความชื่นชอบในการเล่นเกมและซอฟต์แวร์ต่างๆ ทำให้อยากรู้ว่าเทคโนโลยีเหล่านี้ถูกสร้างขึ้นมาได้อย่างไร จึงเริ่มฝึกเขียนโปรแกรมด้วยตนเอง ตั้งแต่ HTML, CSS, JavaScript ไปจนถึง Vue.js, Node.js และ PHP

มุ่งมั่นสู่การเป็นนักพัฒนาที่สามารถนำเทคโนโลยีไปต่อยอดในเชิงธุรกิจ โดยเฉพาะการสร้างเว็บไซต์ E-Commerce และธุรกิจออนไลน์

ปวส. วิทยาลัยเทคนิคขอนแก่น GPAX 3.86 ขอนแก่น E-Commerce Self Learner
